# IFAMDS - Forest Simulation System

## Overview
C++ console-based forest simulation using multiple data structures to monitor and manage forest conditions.

## Features
- Menu-driven interface  
- Arrays for sensor data  
- Linked lists for event tracking  
- Stack/Queue for processing  
- Trees & Graphs for zone management  
- Hashing for fast lookup  
- Algorithms: BFS, DFS, Scheduling  

## Scenarios
Implements 5 forest-based scenarios including fire detection, monitoring, and resource management.

## Output
Shows real-time system updates and state changes in console.
